# Analytics V1 QA Checklist

This checklist is used by QA and engineering to verify the correctness of the metric widgets deployed to the `admin.kovari.in` dashboard. Each widget's backend SQL output must match the corresponding verification query and expected results.

| Metric | Expected Value | Verification Query | Verified | QA Validation Notes |
| :--- | :---: | :--- | :---: | :--- |
| **Total Users** | `16` | `SELECT COUNT(DISTINCT u.id) FROM public.users u WHERE u."isDeleted" = false AND u.id NOT IN (SELECT DISTINCT usr.id FROM public.users usr LEFT JOIN public.profiles prf ON usr.id = prf.user_id JOIN public.admins adm ON LOWER(COALESCE(prf.email, usr.email)) = LOWER(adm.email));` | **YES** | Verifies `users` table exclusion logic. Excludes 4 admins. Includes 1 non-activated user (`yashsabne39@gmail.com`). |
| **Activated Users** | `15` | `SELECT COUNT(DISTINCT u.id) FROM public.users u WHERE u.beta_status = 'activated' AND u."isDeleted" = false AND u.id NOT IN (SELECT DISTINCT usr.id FROM public.users usr LEFT JOIN public.profiles prf ON usr.id = prf.user_id JOIN public.admins adm ON LOWER(COALESCE(prf.email, usr.email)) = LOWER(adm.email));` | **YES** | Verifies `beta_status = 'activated'` filter and admin exclusion. |
| **Returned Users** | `0` | `SELECT COUNT(*) FROM public.users u WHERE u.beta_status = 'activated' AND u."isDeleted" = false AND (u.last_seen_at AT TIME ZONE 'Asia/Kolkata')::date > (u.activation_date AT TIME ZONE 'Asia/Kolkata')::date AND u.id NOT IN (SELECT DISTINCT usr.id FROM public.users usr LEFT JOIN public.profiles prf ON usr.id = prf.user_id JOIN public.admins adm ON LOWER(COALESCE(prf.email, usr.email)) = LOWER(adm.email));` | **NON-FUNC** | **Blocker**: Telemetry gap. `last_seen_at` is only set once at signup, making calendar-day returns mathematically impossible. |
| **Dormant Users** | `4` | `SELECT COUNT(*) FROM public.users u WHERE u.beta_status = 'activated' AND u."isDeleted" = false AND (u.last_seen_at IS NULL OR u.last_seen_at < NOW() - INTERVAL '7 days') AND u.id NOT IN (SELECT DISTINCT usr.id FROM public.users usr LEFT JOIN public.profiles prf ON usr.id = prf.user_id JOIN public.admins adm ON LOWER(COALESCE(prf.email, usr.email)) = LOWER(adm.email));` | **NON-FUNC** | **Blocker**: Telemetry gap. Ticks up automatically to 100% after 7 days post-signup for all users. |
| **Daily Active Users (DAU)** | `0` | `SELECT COUNT(DISTINCT u.id) FROM public.users u WHERE u.beta_status = 'activated' AND u."isDeleted" = false AND u.last_seen_at AT TIME ZONE 'Asia/Kolkata' >= (CURRENT_TIMESTAMP AT TIME ZONE 'Asia/Kolkata')::date AND u.id NOT IN (SELECT DISTINCT usr.id FROM public.users usr LEFT JOIN public.profiles prf ON usr.id = prf.user_id JOIN public.admins adm ON LOWER(COALESCE(prf.email, usr.email)) = LOWER(adm.email));` | **NON-FUNC** | **Blocker**: Telemetry gap. Only counts users on their signup date. |
| **Travel Intent Completed** | `7` | `SELECT COUNT(DISTINCT p.user_id) FROM public.profiles p JOIN public.users u ON p.user_id = u.id WHERE u.beta_status = 'activated' AND u."isDeleted" = false AND p.travel_intentions IS NOT NULL AND jsonb_array_length(p.travel_intentions) > 0 AND u.id NOT IN (SELECT DISTINCT usr.id FROM public.users usr LEFT JOIN public.profiles prf ON usr.id = prf.user_id JOIN public.admins adm ON LOWER(COALESCE(prf.email, usr.email)) = LOWER(adm.email));` | **YES** | Verifies `travel_intentions` JSONB array length > 0. Joins `users` and `profiles` tables. |
| **Interests Sent** | `16` | `SELECT COUNT(*) FROM public.match_interests m WHERE m.from_user_id NOT IN (SELECT DISTINCT usr.id FROM public.users usr LEFT JOIN public.profiles prf ON usr.id = prf.user_id JOIN public.admins adm ON LOWER(COALESCE(prf.email, usr.email)) = LOWER(adm.email));` | **YES** | Counts raw rows in `match_interests` from organic users. Excludes admin-sent test records. |
| **Pending Interests** | `7` | `SELECT COUNT(*) FROM public.match_interests m WHERE m.status = 'pending' AND m.from_user_id NOT IN (SELECT DISTINCT usr.id FROM public.users usr LEFT JOIN public.profiles prf ON usr.id = prf.user_id JOIN public.admins adm ON LOWER(COALESCE(prf.email, usr.email)) = LOWER(adm.email));` | **YES** | Verifies `status = 'pending'` filter. |
| **Accepted Interests** | `0` | `SELECT COUNT(*) FROM public.match_interests m WHERE m.status = 'accepted' AND m.from_user_id NOT IN (SELECT DISTINCT usr.id FROM public.users usr LEFT JOIN public.profiles prf ON usr.id = prf.user_id JOIN public.admins adm ON LOWER(COALESCE(prf.email, usr.email)) = LOWER(adm.email)) AND m.to_user_id NOT IN (SELECT DISTINCT usr.id FROM public.users usr LEFT JOIN public.profiles prf ON usr.id = prf.user_id JOIN public.admins adm ON LOWER(COALESCE(prf.email, usr.email)) = LOWER(adm.email));` | **YES** | Verifies organic-to-organic (stranger) matches are `0`. |
| **Stranger Acceptance Rate** | `0.00%` | `WITH organic AS (SELECT id FROM public.users WHERE id NOT IN (SELECT DISTINCT usr.id FROM public.users usr LEFT JOIN public.profiles prf ON usr.id = prf.user_id JOIN public.admins adm ON LOWER(COALESCE(prf.email, usr.email)) = LOWER(adm.email)) AND "isDeleted" = false) SELECT ROUND(COUNT(CASE WHEN status = 'accepted' THEN 1 END) * 100.0 / NULLIF(COUNT(CASE WHEN status IN ('accepted', 'rejected') THEN 1 END), 0), 2) FROM public.match_interests WHERE from_user_id IN (SELECT id FROM organic) AND to_user_id IN (SELECT id FROM organic);` | **YES** | P0 Dashboard acceptance rate metric. Correctly handles `NULLIF` divide-by-zero guard. |
| **Conversations Started** | `0` | `WITH admin_users AS (SELECT DISTINCT u.id AS admin_id FROM public.users u LEFT JOIN public.profiles p ON u.id = p.user_id JOIN public.admins a ON LOWER(COALESCE(p.email, u.email)) = LOWER(a.email)) SELECT COUNT(*) FROM public.conversations c WHERE c.user_a_id NOT IN (SELECT admin_id FROM admin_users) AND c.user_b_id NOT IN (SELECT admin_id FROM admin_users);` | **YES** | Verifies organic conversations. Checks table `conversations`. |
| **Messages Sent** | `0` | `WITH admin_users AS (SELECT DISTINCT u.id AS admin_id FROM public.users u LEFT JOIN public.profiles p ON u.id = p.user_id JOIN public.admins a ON LOWER(COALESCE(p.email, u.email)) = LOWER(a.email)) SELECT COUNT(*) FROM public.direct_messages d WHERE d.media_type IS DISTINCT FROM 'init' AND d.sender_id NOT IN (SELECT admin_id FROM admin_users) AND d.receiver_id NOT IN (SELECT admin_id FROM admin_users);` | **YES** | Verifies organic messages. Excludes auto-generated `'init'` welcome message. |
| **Notifications Created** | `26` | `SELECT COUNT(*) FROM public.notifications n WHERE n.user_id NOT IN (SELECT DISTINCT usr.id FROM public.users usr LEFT JOIN public.profiles prf ON usr.id = prf.user_id JOIN public.admins adm ON LOWER(COALESCE(prf.email, usr.email)) = LOWER(adm.email));` | **YES** | Verifies total generated notification events for organic users. |
| **Notifications Read** | `2` | `SELECT COUNT(*) FROM public.notifications n WHERE n.is_read = true AND n.user_id NOT IN (SELECT DISTINCT usr.id FROM public.users usr LEFT JOIN public.profiles prf ON usr.id = prf.user_id JOIN public.admins adm ON LOWER(COALESCE(prf.email, usr.email)) = LOWER(adm.email));` | **YES** | Verifies `is_read = true` flag. |
| **Push Attempted** | `2` | `SELECT COUNT(*) FROM public.notifications n WHERE n.push_attempted_at IS NOT NULL AND n.user_id NOT IN (SELECT DISTINCT usr.id FROM public.users usr LEFT JOIN public.profiles prf ON usr.id = prf.user_id JOIN public.admins adm ON LOWER(COALESCE(prf.email, usr.email)) = LOWER(adm.email));` | **YES** | Verifies push notification dispatch logs. |
| **Push Failures** | `0` | `SELECT COUNT(*) FROM public.notifications n WHERE n.push_status = 'failed' AND n.user_id NOT IN (SELECT DISTINCT usr.id FROM public.users usr LEFT JOIN public.profiles prf ON usr.id = prf.user_id JOIN public.admins adm ON LOWER(COALESCE(prf.email, usr.email)) = LOWER(adm.email));` | **YES** | Verifies `push_status = 'failed'` from FCM response telemetry. |
| **No Token Count** | `2` | `SELECT COUNT(*) FROM public.notifications n WHERE n.push_status = 'no_token' AND n.user_id NOT IN (SELECT DISTINCT usr.id FROM public.users usr LEFT JOIN public.profiles prf ON usr.id = prf.user_id JOIN public.admins adm ON LOWER(COALESCE(prf.email, usr.email)) = LOWER(adm.email));` | **YES** | Verifies `push_status = 'no_token'` indicating missing FCM device token. |
